# Cyberpunk 2077 Theme

A collection of VS Code themes inspired by Night City and Johnny Silverhand from Cyberpunk 2077.

## Available Themes

*   **Cyberpunk: Night City** - Inspired by the vibrant fluorescent lights and high-contrast shadows of Night City.
*   **Cyberpunk: Silverhand** - Inspired by the high-octane rebellion of Johnny Silverhand, with red, black, and silver tones.

## Installation

1.  Open Chrome or your browser of choice.
2.  Download the `.vsix` file.
3.  Open VS Code.
4.  Go to Extensions -> "..." -> Install from VSIX...
5.  Select the downloaded file.
6.  Go to File -> Preferences -> Color Theme and select one of the Cyberpunk themes.
